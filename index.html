Handige Linux commando's

Bestands- en mapbeheer

Command Uitleg
ls Toon bestanden in een map
cd {mapnaam} Ga naar map
cd .. Ga een map omhoog
cd Ga naar home map
pwd Toon huidige locatie
mkdir {mapnaam} Maak een nieuwe map
rm {bestandsnaam} Verwijder bestand
cp {bron} {doel} Kopieer bestand
mv {bron} {doel} Verplaats of hernoem bestand

Pakketten installeren

command Uitleg
Sudo apt update Update Ubuntu
sudo apt install {pakketnaam} Installeer pakket

Bekijken & bewerken van bestanden

Command Uitleg
cat {bestandsnaam} Toon de inhoud van bestand
less {bestandsnaam} Toon de inhoud van bestand 1 pagina per keer, zonder hem te openen
nano {bestandsnaam} Edit bestand

Superuser & rechten

Command Uitleg
sudo {command} Voer commando uit als admin
chmod 755 {bestandsnaam} Verander rechten van bestand
chown {user} {bestandsnaam} Verander eigenaar van bestand

Netwerk

Command Uitleg
ip a Toon netwerkinterfaces
ping {adres} Test bereikbaarheid adres
